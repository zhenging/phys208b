1504 Speed Of Transverse Wave

Wave Speed on a String

v=Fμ(15.14)\begin{aligned} v = \sqrt{\frac{F}{\mu}} \quad \bold{(15.14)} \end{aligned} v=μF​​(15.14)​

vvv - Speed of a transverse wave on a string
FFF - Tension in string
μ\muμ - Mass per unit length.

The Speed of Mechanical Waves

v=Restoring force returing the system to equilibriumInertia resisting the return to equilibriumv = \sqrt{\frac{\text{Restoring force returing the system to equilibrium}}{\text{Inertia resisting the return to equilibrium}}} v=Inertia resisting the return to equilibriumRestoring force returing the system to equilibrium​​

The tension FFF in the string plays the role of the restoring force; it tends to bring the string back to its undisturbed, equilibrium configuration. The mass of the string - or, more properly, the linear mass density μ\muμ provides the inertia that prevents the string from returning instantaneously to equilibrium. Hence we have v=Fμv =\sqrt{\dfrac{F}{\mu}}v=μF​​ for the speed of waves on a string.
